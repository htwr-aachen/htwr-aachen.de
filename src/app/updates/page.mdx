## Zum Thema Unterstützung

Ich werde versuchen es leichter zu machen selber Sachen zu schreiben, zu korrigieren etc.

Für Fächer, die jetzt schon da sind also MALO, STOCHA, DATCOM, BUK, SWT ist es ziemlich einfach an den Zusammenfassungen zu arbeiten.
Es gibt im [git repo](https://github.com/jonsch318/htwr-aachen.de) einen Ordner `/teachings`. Hier sind die Sachen in Institute geordnet.
Ein neue Datei wird direkt erkannt und auf der Seite angezeigt.

Die Schreibweise ist einfach Markdown. Es gibt Cheat Sheets im [Internet](https://www.markdownguide.org/cheat-sheet/).
Das allermeiste wird unterstützt.

Ein Update kann dann einfach über ein Pull Request gemacht werden.
Also Forken, Änderungen machen und dann einen Pull Request machen oder Branch machen und dann Pull Request. Muss leider so laufen sonst wird hier scheiße hochgeladen.
