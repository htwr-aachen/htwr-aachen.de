# Nützliche Komponenten

- `/src/layouts`: Layouts. Alle haben eine ähnliche Schnittstelle
  - `/src/layouts/rwth`: Wird von den meisten Lehrstühlen verwendet. Orientiert sich an dem CMS-System der RWTH
- `BasicSubjectInfo`: Links zu üblichen Sachen für ein Fach. z.B. Panikzettel, Folien-Merge, ...
- `SubjectDocumentList`: Tree-View aller Dokumente für ein Fach unter `/public/content-assets/[subject]`
- `SubjectList`: Auswahl zwischen mehreren Fächern.
- `SummaryList`: Listet alle Zusammenfassungen des Fachs
- `SummaryView`: Zeigt eine Zusammenfassung an.
